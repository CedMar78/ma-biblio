# Ma Biblio

Ma Biblio est une application web simple permettant de suivre ses lectures.

Elle permet d’ajouter des livres dans trois catégories :
- À lire
- En cours
- Lus
L’objectif du projet est de créer une première application concrète en HTML, CSS et JavaScript, avec une interface simple, pratique et responsive.

## Fonctionnalités

- Ajouter un livre avec un titre, un auteur, une note personnelle et une couverture
- Classer un livre selon son statut : à lire, en cours ou lu
- Modifier les informations d’un livre
- Changer rapidement le statut d’un livre
- Supprimer un livre
- Sauvegarder les données dans le navigateur grâce au localStorage

## Technologies utilisées

- HTML
- CSS
- JavaScript
- GitHub Pages
